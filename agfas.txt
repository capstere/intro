## 🔍 4) Hur du gör det “100% bevisat” även utanför datainsamlingen (snabb check)

För att avgöra om INF5 **verkligen** saknar policys (och inte bara “saknas i utdraget”), kör dessa på INF5 + INF1 + INF3 och jämför output:

### A) Verifiera policy från *faktisk* registry

```powershell
reg query "HKLM\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate"
reg query "HKLM\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate\AU"
```

### B) Verifiera vilken GPO som sätter Windows Update

```powershell
gpresult /h C:\Temp\gp.html
```

Öppna rapporten och sök på “Windows Update” / “WSUS” / “Specify intranet Microsoft update service location”.

### C) Verifiera update-källa (WSUS vs Microsoft Update)

```powershell
reg query "HKLM\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate" /v WUServer
reg query "HKLM\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate" /v WUStatusServer
reg query "HKLM\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate\AU" /v UseWUServer
```

## ⚠️ Det som avviker (det viktiga!)

### 🔴 INF5 beter sig annorlunda än INF1 och INF3

**Varför?**

### 1) INF5 saknar styrning (i datan)

INF1 och INF3 har tydliga policies:

* Pekar på **WSUS-server**
* Har regler som styr uppdateringar
* Har kontroll på reboot-beteende

INF5:

* De här inställningarna **syns inte i datan**

👉 Troligen mer “fri” Windows Update ➜ mer spontana omstarter [\[danaher-my...epoint.com\]](https://danaher-my.sharepoint.com/personal/labuser_ipt_cepheid_com/Documents/Microsoft%20Copilot%20Chat%20Files/INF5_May10_May14_Correlation_Report.md)

***

### 2) INF1 & INF3 har INTE natt-omstarter

* De visar **inga omstarter 00–02 dessa datum**
* INF5 gör det flera gånger

👉 INF5 har ett **aktivt uppdateringsflöde nattetid**, det har inte de andra [\[danaher-my...epoint.com\]](https://danaher-my.sharepoint.com/personal/labuser_ipt_cepheid_com/Documents/Microsoft%20Copilot%20Chat%20Files/INF5_May10_May14_Correlation_Report.md)

***

### 3) INF5 får problem under uppdatering

* Felkod: **0x800F0984**
* Saknad komponent (telnet.exe)

👉 Detta leder ofta till:

* fler försök
* fler omstarter [\[danaher-my...epoint.com\]](https://danaher-my.sharepoint.com/personal/labuser_ipt_cepheid_com/Documents/Microsoft%20Copilot%20Chat%20Files/INF5_May10_May14_Correlation_Report.md)

***

## 🧠 Enkelt sagt

👉 **INF5 = uppdaterar, krånglar, startar om flera gånger**  
👉 **INF1/INF3 = mer kontrollerade, därför lugnare**
